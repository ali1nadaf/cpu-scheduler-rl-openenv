# cpu-scheduler-rl-openenv
OpenEnv-compliant Reinforcement Learning environment for CPU scheduling optimization. This project simulates a real-world operating system scheduler where an AI agent learns to make optimal process scheduling decisions based on dynamic system states, including arrival time, priority, waiting time, and resource constraints.

The environment supports multiple difficulty levels and realistic workload patterns, including dynamic process injection. It integrates benchmarking against classical scheduling algorithms such as FCFS, Shortest Job First, Priority Scheduling, and Round Robin, enabling direct performance comparison.

A Q-learning agent is implemented to demonstrate adaptive decision-making and learning behavior over time. The project bridges Operating Systems and Artificial Intelligence, showcasing how reinforcement learning can be applied to system-level optimization problems in high-performance computing environments.
